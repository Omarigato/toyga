# Toyga.kz — Дизайн-система v2 (люкс / brand-first)

> Файл для MimoCode: используй как источник истины при генерации токенов Tailwind, CSS-переменных и любых новых компонентов. Ничего не заимствуем из `old/` — старая amber/stone-палитра списывается полностью.

## 0. Тезис бренда

Той — это не «мероприятие», это переход: рождение семьи, совершеннолетие, дом. Toyga.kz оцифровывает не открытку, а **шаңырақ** — купольный обод юрты, символ семьи и очага (он же на флаге Казахстана). Приглашение должно ощущаться как личная восковая печать на плотной бумаге, а не как маркетинговый лендинг со свадебными стоками.

**Единственная работа страницы-приглашения:** гость открывает ссылку на телефоне → за 2 секунды считывает «кто, когда, куда» → одним касанием подтверждает приход. Всё остальное — атмосфера вокруг этой задачи, не поверх неё.

**Что запрещено:** кремовый фон + терракотовый акцент (#F4F1EA/#D97757 — клише AI-дизайна), общие свадебные стоки (кольца, лепестки роз, курсивный "Great Vibes"), плоские караван-сарай орнаменты как текстура фона на весь экран.

## 1. Палитра

Тёмная бархатно-войлочная база + состаренное золото. Работает и как «конверт» (тёмный режим по умолчанию для публичных приглашений), и как светлая «бумага» (кабинет организатора).

| Токен | HEX | Роль |
|---|---|---|
| `qara-ink` | `#15110D` | Основной тёмный фон (приглашение, hero, footer) |
| `ac-parqyt` | `#F4ECD8` | Тёплая слоновая кость — светлые поверхности, карточки, «бумага» |
| `saukele-gold` | `#B8902E` | Основной акцент — состаренная латунь/золото (CTA, обводки, иконки) |
| `torgyn-wine` | `#5C1A24` | Вторичный акцент — глубокий бархатный бордо (бейджи статусов, hover-состояния, второй CTA) |
| `boz-dala` | `#9C9280` | Нейтральный степной серо-бежевый — второстепенный текст, границы, плейсхолдеры |
| `kok-tengri` | `#26424A` | Третичный акцент — глубокий тенгри-тил (ссылки на светлом фоне, инфо-состояния, мужская/групповая роль в RSVP) |

Правила использования:
- `saukele-gold` — только для акцентов и интерактива, никогда как заливка больших площадей (иначе превращается в «золотой Comic Sans»). Целевая доля золота на экране ≤ 8%.
- На `qara-ink` текст — `ac-parqyt` при 92% непрозрачности (не чистый белый — от него бренд «дешевеет»).
- Статусы RSVP: подтверждено → `saukele-gold` заливка; не определился → `boz-dala` контур; отказ → `torgyn-wine` контур (не красный/зелёный светофор — это разрушает бренд).

CSS-переменные (добавить в `frontend/src/app/globals.css` вместо amber/stone):

```css
:root {
  --color-ink: #15110D;
  --color-parchment: #F4ECD8;
  --color-gold: #B8902E;
  --color-wine: #5C1A24;
  --color-steppe: #9C9280;
  --color-tengri: #26424A;
}
```

## 2. Типографика

Контент двуязычный/трёхъязычный (kk / ru / en) — **обязательно** проверять полный набор казахских расширенных символов: `Ә ә Ғ ғ Қ қ Ң ң Ө ө Ұ ұ Ү ү Һ һ І і` в каждом выбранном шрифте перед подключением.

| Роль | Шрифт | Источник | Где использовать |
|---|---|---|---|
| Display (декоративный) | **Yeseva One** | Google Fonts | Имена молодожёнов в hero, крупные заголовки секций, число на countdown. Использовать скупо — не более 1–2 экземпляров на экран |
| Body (основной) | **Golos Text** | Google Fonts | Весь UI-текст, формы, кнопки, параграфы, навигация |
| Utility (техническая/цифры) | **PT Mono** | Google Fonts | Даты, время, RSVP-код, счётчик гостей, суммы — «выгравированная табличка» |
| Eyebrow (лейбл, опционально) | **Forum** (капс, трекинг +0.08em) | Google Fonts | Надсекционные лейблы: «ЦЕРЕМОНИЯ», «АДРЕС», «ПОДАРКИ» |

Масштаб (mobile-first, `rem`):

```
--text-display-xl: 3.25rem/1.05   /* имена в hero */
--text-display-lg: 2.25rem/1.1    /* заголовки секций */
--text-display-md: 1.5rem/1.2
--text-body-lg: 1.125rem/1.5
--text-body-md: 1rem/1.55
--text-body-sm: 0.875rem/1.5
--text-eyebrow: 0.75rem/1.4, tracking 0.08em, uppercase
--text-mono: 1rem/1.4 (PT Mono)
```

Правило пары: Yeseva One появляется **только** на тёмном `qara-ink` фоне с золотым или parchment цветом — на светлых карточках дашборда заголовки идут через Golos Text 600 weight, чтобы декоративный шрифт не превратился в шум админки.

## 3. Форма, тень, движение

- **Радиусы:** карточки/модалки `1.25rem` (`rounded-2xl`), кнопки `0.75rem`, «печать»/бейджи — круг. Отказ от `rounded-full` на кнопках (это уже generic-паттерн amber-версии).
- **Тени:** без цветных amber-теней. Одна фирменная тень — `shadow-seal`: `0 8px 30px -8px rgba(21,17,13,0.45)`, для тёмных карточек — `0 1px 0 rgba(184,144,46,0.25) inset` (тонкая золотая грань сверху, как оклад).
- **Границы:** `1px solid rgba(184,144,46,0.18)` на карточках вместо серых бордеров — ощущение оклада, не UI-кита.
- **Motion:** одно оркестрованное движение на страницу, не россыпь micro-interactions. Основная последовательность при заходе на приглашение: печать (см. §4) раскрывается → имена проявляются snap-fade (220ms, ease-out) → остальной контент по scroll-reveal (translateY 12px→0, 400ms, stagger 60ms). Уважать `prefers-reduced-motion`: печать сразу в открытом состоянии, без stagger.

## 4. Фирменный элемент — «Печать Шаңырақ»

Единственный узнаваемый визуальный якорь бренда: круглая линейная эмблема — купол шаңырақа, вписанный в контур, с завитком **қошқар-мүйіз** (бараний рог) по кромке. Тонкая золотая линия на `qara-ink`.

Используется четырежды и больше нигде:
1. **Экран открытия приглашения** — печать «раскрывается» как диафрагма/сургучная печать (не спиннер-кружок).
2. **Прогресс скролла** — тонкое кольцо вокруг плавающей кнопки «RSVP» на приглашении, заполняется по мере скролла.
3. **Разделитель секций** — маленький центрированный глиф печати вместо горизонтальной линии между блоками контента.
4. **Watermark в футере и favicon/OG-карточке.**

Больше нигде орнамент не используется как фоновая текстура на весь экран — это единственное место, где бренд «берёт риск», всё остальное вокруг должно быть тихим.

## 5. Что переносим из старой палитры → новую (маппинг для миграции)

| Было (amber/stone) | Стало |
|---|---|
| `from-amber-600 to-amber-500` (primary button) | `bg-[var(--color-gold)] text-[var(--color-ink)]` |
| `stone-900 / stone-100` текст | `qara-ink` / `ac-parqyt` |
| `.glass` (белая подложка 0.8 alpha) | `.glass` → `rgba(21,17,13,0.55)` + `backdrop-blur(16px)` + золотая обводка 1px |
| `.gradient-text` amber→amber | `linear-gradient(135deg, var(--color-gold), #E8C978)` только для чисел/цифр countdown, не для длинных заголовков |
| focus-ring `amber-500` | focus-ring `saukele-gold`, offset на `qara-ink` фоне — offset-color `ac-parqyt` |
